Colorizing the Prokudin-Gorskii Photo Collection

Shiva Peri - Assignment 1

WINDOW ALGORITHM

  • 100px image window considered
  • Used recursive multiscale image pyramid (scale factor 0.3)
  • Base case: window size < 400px
  • +/- 15px displacements used
  • Minimized structural similarity index measure (SSIM) as image distance metric.
  • considered weighted loss of RGB SSIM (weight 0.1) and gradient SSIM (weight 1.0)
